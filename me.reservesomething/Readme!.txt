Модуль Me.ReserveSomething 
v1.0 
08 октября 2020 г.

Что нового
----------
*  Простой модуль, у которого есть в административной части два настраиваемых параметра: список из 3 элементов и строка. 
*  Имеется простой компонент для вывода на экран параметров настроек. 
*  Настройки компонента должны сохраняться. 

Инсталляция
-----------
 Через админку (Раздел Marketplace) штатно ставим модуль Me.ReserveSomething

Деинсталляция
-------------
 Через админку (Раздел Marketplace) штатно деинсталлируем модуль Me.ReserveSomething
